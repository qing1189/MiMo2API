如何获取 cURL?
  1. 登录 aistudio.xiaomimimo.com
  2. 按 F12 → Network 标签
  3. 发一条消息，找到 chat 请求
  4. 右键 → Copy as cURL，粘贴到下方
从浏览器 Cookie 中获取以下三个值：
F12 → Application → Cookies → aistudio.xiaomimimo.com
或直接粘贴整段 Cookie 字符串（用 ; 分隔），点"解析填充"自动拆分
调度规则说明
  • 默认权重 100，按权重加权随机分配请求
  • 401/403 认证失败 → 权重降到 10，冷却 10 分钟
  • 429 限流 → 权重减半，冷却 5 分钟
  • 5xx / 超时 / 网络错误 → 权重 -20，冷却 1 分钟
  • 冷却结束后按 +10/分钟 线性恢复到 100
  • 连续成功时每次回升 5 点，快速退出降级状态
  • 所有账号都冷却时会自动选最早恢复的那个（降级保底）
